# Project Scope

## Insurance Type Decision

The platform is designed to support multiple insurance policy types in the future through the `policy_type` field.

For the first version of the internship project, only one insurance type will be fully implemented:

- Health Insurance

Other insurance types will be shown as coming soon:

- Vehicle Insurance
- Life Insurance
- Travel Insurance
- Property Insurance

## Reason

This keeps the first version focused and achievable while leaving the database and backend structure flexible for future expansion.

## Current Backend Setting

The active policy type is configured using:

```text
ACTIVE_POLICY_TYPE="Health Insurance"
```
